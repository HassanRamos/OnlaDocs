Client User Guide

Restaurant operations documentation for admins, managers, waiters, and cashiers.

This documentation explains how to use the main restaurant management modules: menus, tables, reservations, POS, orders, customers, staff, payments, reports, settings, shifts, assignments, and due-payment reporting.
