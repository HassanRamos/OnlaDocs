Client-facing documentation package

Restaurant Management System User Guide

This static documentation site is organized according to the menu structure visible in your installed system: restaurant operations, POS, tables, payments, reports, cash register, kitchen, hotel, inventory and system settings.

Start with Quick GuideOpen Documentation Plan

How to use this package

  1. Open index.html in a browser to review the documentation locally.
  2. Replace screenshot placeholders with screenshots from your installed SoraHub restaurant system.
  3. Upload this folder to your hosting account, cPanel public_html folder, subdomain, or documentation site.
  4. Use the Print / Save PDF button on any page to produce printable handouts.

Important editing note

The pages are written for your system workflow and menu layout. Before sharing with clients, add your real screenshots and adjust any wording where your installed version uses a different button name, permission, branch setting or enabled module.
