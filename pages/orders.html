Orders

The Orders page is used to track all created orders, review status, open order details, manage KOT progress, cancel orders when permitted and follow the order lifecycle.

daily use service

Where to find it

Open the left sidebar and go to Front of House → Orders. If the option is not visible, the logged-in user may not have permission or the module may be disabled for the account.

Before you start

  • Confirm you are working in the correct branch/outlet.
  • Make sure your user role has permission to view or edit this area.

Step-by-step guide

  • Open Orders from the left menu.
  • Use filters such as date, status, customer, order type, table or branch.
  • Open an order to view items, quantity, taxes, discount, payment status and KOT status.
  • Update order status according to the restaurant workflow.
  • Use cancellation or refund actions only when authorized.
  • Use reports for end-of-day order reconciliation.
