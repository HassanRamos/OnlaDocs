Automation

Daily Menu PDF Email

Send a daily PDF copy of the restaurant menu to the administrator at a scheduled time.

What this module is forMain use casesTypical workflowImportant notesScreens / Visual Guide

What this module is for

  • This feature automatically prepares a menu PDF and emails it to the administrator every day.
  • It helps owners and managers review the menu, prices, and item availability without manually exporting the menu.

Main use cases

  • Daily menu review.
  • Price confirmation.
  • Item availability monitoring.
  • Internal reporting.
  • Sharing the latest menu with partners or staff.

Typical workflow

  1. Open Email Settings.
  2. Enable the Menu PDF Mail option.
  3. Set the time the email should be sent.
  4. Save the settings.
  5. Confirm that SMTP/email configuration is working.
  6. Check the administrator email for the PDF at the scheduled time.

Important notes

  • The feature requires working email/SMTP configuration.
  • The email is sent only when the feature is enabled.
  • The scheduled time follows the system or server timezone, so confirm timezone settings first.
