Restaurant Menu Module

Create menus, menu items, and item categories so customers and staff can quickly find the right items during ordering.

What this module is forMain featuresTypical workflowGood practiceScreens / Visual Guide

What this module is for

  • Use this module to organize the food and drinks offered by the restaurant.
  • A restaurant can maintain multiple menus such as Breakfast, Lunch, Dinner, Drinks, Specials, or Seasonal Menu.
  • Each menu can contain menu items, and each item can be grouped under categories for easier filtering.

Main features

  • Add, edit, and remove menus.
  • Create menu items under the correct menu.
  • Use item fields such as name, price, description, category, availability, and other details provided by your installation.
  • Filter menu items from the menu-item listing screen.
  • Create item categories and attach menu items to those categories.

Typical workflow

  1. Open the Menu area from the sidebar.
  2. Click Add Menu to create a new menu group.
  3. Open Menu Items and click Add Menu Item.
  4. Fill in the item details and select the menu/category where the item belongs.
  5. Save the item and confirm that it appears in the correct list.
  6. Use edit/delete actions when you need to update old items or remove unavailable entries.

Good practice

  • Use clear item names that staff and customers can understand easily.
  • Group items into meaningful categories such as Starters, Main Meals, Drinks, Desserts, or Combos.
  • Deactivate or update unavailable items instead of leaving customers to order items the kitchen cannot prepare.
  • Review prices before making menus visible to customers.
